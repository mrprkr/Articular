Events Tile Demo


This demo shows the functionality and visual design of the events tiles. The design has considered a visually led approach and how the information is presented on a range of mobile and desktop devices.


Colour is used to differentiate states, with plenty of white space around vital event information to present in a clear and consise way. The date is the start date of the event; open/closed communicates the status of event entry, with both "read more" and "view event" taking the user to the event details page.


The "show more events" button allows the user to load more events to the view, simillar to pagination. There is a subtle fade as events are loaded and displayed.
